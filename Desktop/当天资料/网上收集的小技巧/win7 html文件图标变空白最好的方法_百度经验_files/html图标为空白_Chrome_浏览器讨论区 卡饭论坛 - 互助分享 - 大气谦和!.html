如果设置IE为默认浏览器，图标显示正常。
卸载重装过chrome，重装完当时图标显示也正常，但一重启就变成空白的了。这是怎么回事啊？

解决方法：(原文地址http://www.cnblogs.com/hanleytowne/archive/2013/04/28/3048836.html)
1.进入注册表(regedit)

2.进入下面目录
HKEY_CLASSES_ROOT\Wow6432Node\CLSID\{42042206-2D85-11D3-8CFF-005004838597}/Old Icon

进入上面这个目录后，会看到下面3项：
ChromeHTML,htmlfile,mhtmlfile ，现在要做的设置ChromeHTML下的defaultIcon的值。

3.设置方法
如果ChromeHTML没有defaultIcon，在ChromeHTML右键-->新建-->项，将新建的项名称修改为: defaultIcon
然后选中刚刚新建的 defaultIcon ,在右边视图中，右键修改值，将值设置为: [Chrome安装目录的chrome.exe,并添加逗号，加上数值0]，比如我的设置是：

C:\Program Files (x86)\Google\Chrome\Application\chrome.exe,0
设置好后，去刷新下有html目录的文件，熟悉的chrome的图标就出现了。
