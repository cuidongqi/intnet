中文man的安装

[日期：2014-03-06] 来源：Linux社区  作者：zengweidao [字体：大 中 小]

首先需要确认的是有没有安装中文支持，如果没有请安装：
#yum groupinstall -y "Chinese Support"

现在下载安装包:

wget http://manpages-zh.googlecode.com/files/manpages-zh-1.5.1.tar.gz  (这个wget是直接网页下载的一个命令，没有的话需要你安装下yum install -y wget)

操作步骤如下:

#tar zxvf manpages-zh-1.5.1.tar.gz   

#cd manpages-zh-1.5.1

#./configure --prefix=/usr/local/zhman --disable-zhtw             

#make（有些版本是没有这个命令的，需要自己安装）

#make install

以上安装完成.请做如下操作:

#cd ~

#vi .bashrc

在.bashrc中增加:
alias cman='man -M /usr/local/zhman/share/man/zh_CN'       

#source .bashrc          //在命令行执行，为了使刚刚添加的alias生效

OK!现在就可以使用中文版的man了，在这里是cman，此时，你发现你可以用自带的man，还有安装有中文的cman了，是不是很爽呀。。。
如:

cman cd 

cman ls

如果不行请更改#vim /etc/sysconfig/i18n（里面内容清空）增加-->
LANG="zh_CN.GB18030"
LANGUAGE="zh_CN.GB18030:zh_CN.GB2312:zh_CN"
SUPPORTED="zh_CN.UTF-8:zh_CN:zh:en_US.UTF-8:en_US:en"
SYSFONT="lat0-sun16"
